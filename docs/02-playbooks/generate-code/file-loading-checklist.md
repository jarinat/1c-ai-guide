# Чеклист: загрузка данных из файлов

Дополнительный чеклист для задач типа «загрузить данные из Excel/CSV в объекты БД».
Применяется совместно с основным [чеклистом генерации кода](generation-checklist.md).

## Чтение файла
- [ ] Файл передаётся на сервер через временное хранилище (см. [INT-REC-001](../../01-standards/integrations/recommended/excel-loading-pattern.md))
- [ ] Временный файл удаляется после чтения (в том числе в ветке Исключение)
- [ ] Маппинг колонок файла → программные имена вынесен в отдельную функцию

## Приведение типов
- [ ] Типы данных из файла приведены к типам БД перед использованием в запросах (см. [DAO-REC-001](../../01-standards/data-and-objects/recommended/excel-type-casting.md))
- [ ] Числовые артикулы/номера приведены к строкам с `Формат(, "ЧГ=")`
- [ ] Дробные значения (размеры) приведены с правильным разделителем `Формат(, "ЧРД=.; ЧГ=")`
- [ ] Даты приведены к нужному типу (`Дата` / `ДатаВремя`)

## Сопоставление с БД
- [ ] Поиск ссылок по натуральным ключам выполняется пакетным запросом, а не в цикле (см. [QRY-FORBIDDEN-001](../../01-standards/queries/forbidden/no-queries-in-loop.md))
- [ ] Поиск по натуральным ключам учитывает возможность дублей (добавить диагностический пакет `ИМЕЮЩИЕ КОЛИЧЕСТВО(РАЗЛИЧНЫЕ) > 1`)
- [ ] Если JOIN идёт на несколько таблиц БД — рассмотрено разделение на этапы с `ВЫБРАТЬ РАЗЛИЧНЫЕ` (см. [QRY-REC-001](../../01-standards/queries/recommended/batch-query-with-distinct-temp-tables.md))
- [ ] Ненайденные записи (номенклатура, документы) выявляются и логируются

## Обработка и запись
- [ ] Есть dry-run режим (проверка без записи)
- [ ] Ошибка обработки одной записи не останавливает обработку остальных (`Попытка/Исключение` внутри цикла)
- [ ] Идемпотентность: повторный запуск безопасен (пропуск записей, где значение уже совпадает)
- [ ] Итоговая сводка со счётчиками: обработано, заполнено, пропущено, ошибки

## Безопасность записи
- [ ] Если запись с `ОбменДанными.Загрузка = Истина` — обосновано, почему безопасно пропускать подписки
- [ ] Если запись обычная — учтены проверки и подписки, которые могут сработать
