# Playbook: генерация кода

## Порядок работы
1. Определить тип объекта и тип задачи.
2. Определить применимые тематические разделы.
3. Прочитать обязательные правила.
4. Проверить запрещённые практики.
5. Учесть рекомендуемые практики.
6. Сгенерировать код.
7. Выполнить самопроверку.
