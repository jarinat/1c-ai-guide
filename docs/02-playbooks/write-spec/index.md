# Playbook: написание ТЗ

## Цель
Сформировать понятное и детализированное ТЗ для AI с учетом правил разработки на 1С.

## Порядок работы
1. Определить бизнес-цель задачи.
2. Определить затрагиваемые объекты 1С.
3. Найти применимые разделы правил.
4. Зафиксировать ограничения.
5. Описать ожидаемую реализацию.
6. Описать проверки и критерии приемки.
