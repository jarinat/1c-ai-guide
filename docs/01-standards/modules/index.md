# Модули

## Назначение раздела
Правила по структуре, оформлению и содержимому модулей 1С: комментирование, организация кода, области, экспортные методы.

## Подразделы
- `mandatory/` — обязательные правила
- `recommended/` — рекомендуемые практики
- `forbidden/` — запрещённые практики
- `exceptions/` — допустимые исключения

## Что сюда не входит
- Клиент-серверное разделение и директивы компиляции — см. `client-server/`
- Именование переменных и процедур — см. `naming/`
- Формы и обработчики событий — см. `forms/`
