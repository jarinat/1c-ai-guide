# Интеграции

## Назначение раздела
Правила по интеграциям: загрузка/выгрузка файлов (Excel, CSV), обмен данными, работа с внешними системами и API.

## Подразделы
- `mandatory/` — обязательные правила
- `recommended/` — рекомендуемые практики
- `forbidden/` — запрещённые практики
- `exceptions/` — допустимые исключения

## Что сюда не входит
- Работа с данными и объектами внутри БД — см. `data-and-objects/`
- Язык запросов — см. `queries/`
- Формы и клиент-серверное взаимодействие — см. `forms/`, `client-server/`
