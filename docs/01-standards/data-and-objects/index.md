# Данные и объекты

## Назначение раздела
Правила по работе с данными, объектами БД, таблицами значений и приведению типов в 1С:Предприятии. Сюда входит чтение/запись данных, преобразование типов, работа с коллекциями и табличными частями.

## Подразделы
- `mandatory/` — обязательные правила
- `recommended/` — рекомендуемые практики
- `forbidden/` — запрещённые практики
- `exceptions/` — допустимые исключения

## Что сюда не входит
- Язык запросов и оптимизация запросов — см. `queries/`
- Клиент-серверное разделение — см. `client-server/`
- Интеграции и обмен данными — см. `integrations/`
