# QRY-FORBIDDEN-001. Не выполнять запросы в цикле по умолчанию

## Статус
Проверено

## Уровень
Запрещенная практика

## Область
Запросы

## Краткое правило
Не выполняй один и тот же или похожие запросы внутри цикла, если задачу можно решить одним запросом, пакетной обработкой или другой архитектурой без запросов в цикле.

## Зачем это нужно
Многократное выполнение запросов в цикле обычно создает лишнюю нагрузку на платформу и базу данных, ухудшает производительность и скрывает более удачную архитектуру. Чаще всего такие конструкции можно заменить одним запросом по набору значений, временными таблицами, соединениями или предварительной выборкой данных.

## Как применять
1. При написании ТЗ, проектировании архитектуры и генерации кода исходно выбирай решение без запросов в цикле.
2. Сначала проверяй, можно ли получить нужные данные одним запросом по набору значений, через временные таблицы, соединения, пакетную обработку или предварительное чтение данных.
3. Если в коде уже есть запрос в цикле, в ревью и при поиске ошибок считай это нарушением по умолчанию, пока не подтверждено исключение.
4. Не предлагай и не фиксируй запросы в цикле без применения исключения [QRY-EXCEPTION-001](../exceptions/queries-in-loop-by-explicit-approval.md).

## Как проверять в review
- Ищи конструкции, где `Новый Запрос`, `Запрос.Текст`, `Запрос.Выполнить()` или `Выбрать()` находятся внутри `Для каждого`, `Пока`, `Для`, обхода коллекций или аналогичного цикла.
- Проверяй, нельзя ли заменить серию однотипных запросов одним запросом с отбором `В (&МассивЗначений)` или другой пакетной схемой.
- Если автор решения считает запрос в цикле оправданным, требуй явное подтвержденное исключение и его документирование.

## Пример правильно
```bsl
Запрос = Новый Запрос;
Запрос.Текст =
"ВЫБРАТЬ
|    Остатки.Номенклатура,
|    Остатки.КоличествоОстаток
|ИЗ
|    РегистрНакопления.ТоварыНаСкладах.Остатки(
|        ,
|        Номенклатура В (&МассивНоменклатуры)
|    ) КАК Остатки";

Запрос.УстановитьПараметр("МассивНоменклатуры", МассивНоменклатуры);
Результат = Запрос.Выполнить().Выбрать();
```

## Пример неправильно
```bsl
Для каждого ЭлементНоменклатуры Из МассивНоменклатуры Цикл
    Запрос = Новый Запрос;
    Запрос.Текст =
    "ВЫБРАТЬ
    |    Остатки.Номенклатура,
    |    Остатки.КоличествоОстаток
    |ИЗ
    |    РегистрНакопления.ТоварыНаСкладах.Остатки(
    |        ,
    |        Номенклатура = &Номенклатура
    |    ) КАК Остатки";

    Запрос.УстановитьПараметр("Номенклатура", ЭлементНоменклатуры);
    Результат = Запрос.Выполнить().Выбрать();
КонецЦикла;
```

## Исключения
Допустимо только по [QRY-EXCEPTION-001](../exceptions/queries-in-loop-by-explicit-approval.md).
