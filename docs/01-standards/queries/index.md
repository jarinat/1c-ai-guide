# Запросы

## Назначение раздела
Здесь хранятся канонические правила по проектированию и использованию запросов в 1С:Предприятии: структура запросов, границы допустимого применения, типовые ошибки производительности и согласованные исключения.

## Подразделы
- `mandatory/` — обязательные правила
- `recommended/` — рекомендуемые практики
- `forbidden/` — запрещённые практики
- `exceptions/` — допустимые исключения

## Что сюда не входит
- клиент-серверное разделение и директивы компиляции относятся к `client-server/`
- правила по формам относятся к `forms/`
- общие правила по модулям и структуре кода относятся к `modules/`
- правила по объектной модели и работе с данными вне языка запросов относятся к `data-and-objects/`
