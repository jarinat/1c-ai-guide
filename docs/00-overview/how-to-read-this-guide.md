# Как читать этот гид

## Для людей
1. Сначала открой `docs/01-standards/index.md`.
2. Затем перейди в нужный тематический раздел.
3. Внутри раздела сначала смотри `mandatory/`, потом `forbidden/`, потом `recommended/`, затем `exceptions/`.

## Для AI-агентов
1. Определи тип задачи.
2. Выбери соответствующий playbook.
3. Определи тематический раздел стандартов.
4. Примени правила в установленном порядке приоритетов.
